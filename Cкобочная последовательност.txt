Дана скобочная последовательность: [ ( (()) () (()) ] ].
1.  Можно ли считать эту последовательность правильной?
Нет, нельзя.
2. Чтобы эта последовательность стала правильно, необходимо:
либо 1) удалить одну "]" и одну "(" получится => [ (()) () (()) ],
либо 2) добавить одну "[" и одну ")" получится => [ [ ( (()) () (()) ) ] ].
